roundRTER Documentation

Gestion des arrondis

Description

La première décimale non significative égale à 0, 1, 2, 3 ,4 ou 5 n’incrémente pas la décimale significative. La première décimale non significative égale à 6, 7, 8 ou 9 incrémente la décimale significative.

Usage

roundRTE(r, accuracy = 0.1)

Arguments

r

number to be rounded
